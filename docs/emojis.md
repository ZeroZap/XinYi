在 GitHub 的 Markdown 中使用 Unicode 图标确实能让文档更生动。这些图标无需外部资源，直接复制粘贴即可使用。以下是获取途径和常用图标：

## 📍 获取途径

1. **直接复制使用** - 下面列出常用图标，直接复制粘贴
2. **专门网站查找**：
   - [Emojipedia](https://emojipedia.org/) - 最全的 emoji 查找
   - [Unicode Table](https://unicode-table.com/) - 按分类浏览所有 Unicode 字符
   - [Get Emoji](https://getemoji.com/) - 快速复制粘贴
3. **系统自带**：
   - Windows: `Win + .` 或 `Win + ;` 打开表情面板
   - Mac: `Ctrl + Cmd + Space`
   - Linux: 大多数桌面环境有字符映射表

## ⚠️ 注意事项

- 不同系统显示可能略有差异
- 建议在 README、文档中使用，避免在核心代码中过度使用
- 保持一致性，避免过多不同类型的图标混合
